## Phase 1: Early Season (Weeks 1-4)

### Goals
- Transition from indoor to outdoor
- Rebuild bike handling confidence
- Establish outdoor FTP baseline
- Build volume gradually

### Weekly Template

| Day | Session | Duration | Notes |
|-----|---------|----------|-------|
| Mon | Rest | - | - |
| Tue | Intervals (indoor option) | 75 min | Sweet spot or tempo |
| Wed | Easy spin or rest | 45-60 min | Recovery |
| Thu | Outdoor tempo | 90 min | Terrain practice |
| Fri | Rest | - | - |
| Sat | Long outdoor | 3-4 hr | Group ride or solo endurance |
| Sun | Easy outdoor | 1.5-2 hr | Skills focus |

### Key Workouts

**Tuesday - Indoor/Outdoor Sweet Spot:**
```
Outdoor version:
- Find 15-20 min climb or flat section
- 3×15 min @ 88-92% FTP
- 5 min easy between intervals
```

**Thursday - Outdoor Tempo with Terrain:**
```
Route: Rolling terrain
- 60 min @ 75-85% FTP average
- Let power vary with terrain
- Practice pacing by feel
```

**Saturday - Long Ride:**
```
Week 1-2: 2.5-3 hr, mostly Z2
Week 3-4: 3-4 hr, include tempo sections
- Join group ride at appropriate level
- OR solo with café stop
```

### Week 4: FTP Test
- Outdoor 20-min test on familiar climb/route
- Compare to indoor FTP
- Adjust zones for outdoor training

## Phase 2: Build 1 (Weeks 5-8)

### Goals
- Develop threshold power
- Climbing-specific fitness
- Increase training load

### Weekly Template

| Day | Session | Duration | Notes |
|-----|---------|----------|-------|
| Mon | Rest | - | - |
| Tue | VO2max or threshold | 75-90 min | Key workout #1 |
| Wed | Easy endurance | 60-90 min | Active recovery |
| Thu | Climbing intervals | 90-120 min | Key workout #2 |
| Fri | Rest or easy | 0-45 min | Pre-weekend recovery |
| Sat | Long with intensity | 4-5 hr | Race simulation |
| Sun | Moderate endurance | 2-3 hr | Aerobic maintenance |

### Key Workouts

**Tuesday - Threshold Development:**
```
Week 5: 3×10 min @ 100% FTP
Week 6: 2×15 min @ 100% FTP
Week 7: 3×12 min @ 100% FTP
Week 8: 2×20 min @ 95% FTP (recovery week)
```

**Thursday - Hill Repeats:**
```
Find: 6-10 min climb at 5-8%

Week 5: 4×6 min @ 95% FTP
Week 6: 5×6 min @ 97% FTP
Week 7: 4×8 min @ 95% FTP
Week 8: 3×6 min @ 90% FTP (recovery)
```

**Saturday - Long Ride with Race Simulation:**
```
Structure:
- 90 min easy warm-up
- 60-90 min race-pace effort (group or solo)
- 60-90 min easy cool-down

Include:
- Attacks and chase efforts
- Variable terrain
- Nutrition practice
```

## Phase 3: Build 2 (Weeks 9-12)

### Goals
- Peak aerobic capacity (VO2max)
- Race-specific fitness
- Tactical preparation

### Weekly Template

| Day | Session | Duration | Notes |
|-----|---------|----------|-------|
| Mon | Rest | - | - |
| Tue | VO2max intervals | 75 min | Key workout #1 |
| Wed | Easy spin | 60 min | Recovery |
| Thu | Race simulation | 90-120 min | Key workout #2 |
| Fri | Rest | - | - |
| Sat | Racing or hard group | 3-4 hr | Race or simulation |
| Sun | Easy endurance | 2-2.5 hr | Recovery |

### Key Workouts

**Tuesday - VO2max Development:**
```
Week 9: 5×3 min @ 110% FTP (3 min recovery)
Week 10: 6×3 min @ 110% FTP
Week 11: 5×4 min @ 108% FTP
Week 12: 4×3 min @ 105% FTP (recovery)
```

**Thursday - Race Simulation:**
```
Options:
A) Local training race
B) Hard group ride with attacks
C) Solo simulation:
   - 20 min @ 85% (sitting in)
   - 3 min @ 120% (attack)
   - 10 min @ 95% (chase)
   - Repeat 2-3x
```

**Saturday - Race or Group Ride:**
```
If racing: Follow race plan
If group ride: Treat as race
- Attack on climbs
- Contest sprints
- Practice positioning
```

## Weather Contingencies

| Conditions | Adjustment |
|------------|------------|
| Rain (light) | Outdoor OK, reduce intensity 5% |
| Rain (heavy) | Move to indoor trainer |
| Extreme heat | Early morning or indoor |
| Strong wind | Adjust route, expect lower power |
| Cold (<5°C) | Extended warm-up, layer appropriately |
